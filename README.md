# Лабораторна робота №2: DevOps та автоматизація

Цей репозиторій містить результати виконання лабораторної роботи з налаштування LVM, роботи зі стандартними потоками Linux та автоматизації розгортання ПЗ за допомогою Ansible.

## Склад проекту
* [cite_start]`task7.sh` — скрипт, що демонструє роботу зі стандартними потоками (stdin, stdout, stderr) та кодами завершення[cite: 353].
* [cite_start]`inventory.ini` — файл інвентаризації з IP-адресою керованого хоста[cite: 290].
* [cite_start]`playbook.yml` — Ansible-плейбук для віддаленого встановлення Nginx та PHP[cite: 355, 358].

## Вимоги
* [cite_start]Операційна система: Linux Mint (або інша на базі Debian)[cite: 234].
* [cite_start]Встановлений Ansible на основній машині[cite: 354].
* [cite_start]Налаштований SSH-доступ за ключами до віддаленого хоста[cite: 352].
