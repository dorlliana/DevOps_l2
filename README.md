# Лабораторна робота №2: DevOps та автоматизація

Цей репозиторій містить результати виконання лабораторної роботи з налаштування LVM, роботи зі стандартними потоками Linux та автоматизації розгортання ПЗ за допомогою Ansible.

## Склад проекту
* `task7.sh` — скрипт, що демонструє роботу зі стандартними потоками (stdin, stdout, stderr) та кодами завершення.
* `inventory.ini` — файл інвентаризації з IP-адресою керованого хоста.
* `playbook.yml` — Ansible-плейбук для віддаленого встановлення Nginx та PHP.

## Вимоги
* Операційна система: Linux Mint (на базі Debian).
* Встановлений Ansible на основній машині.
* Налаштований SSH-доступ за ключами до віддаленого хоста.

## Як використовувати
1. Переконайтеся, що керований хост (`_host`) увімкнений та доступний по мережі.
2. Запустіть автоматизацію командою:
   ```bash
   ansible-playbook -i inventory.ini playbook.yml -K
